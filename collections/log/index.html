Reitit for Windows Phone 8.1

Tue, Aug 26, 2014 Over the summer I have been developing a Windows Phone 8.1 version of my Helsinki metropolitan area route planner app, Reitit. New features include: GPS tracking on the map A new more streamlined route listing Advanced search options The new version is already mostly feature complete. However, the release of the app will be delayed due to problems I have encountered with the new framework: The app is too slow. Page navigation is noticeably slower than in the old version.
